# PURSUE UAP Archive Explorer

Interactive bilingual (中文/EN) explorer for all 294 records of the U.S. Department of War's
PURSUE declassified UAP releases (2026-05-08 / 05-22 / 06-12).

- Charts & statistics (agency / type / decade / score / theme / redaction)
- Filterable browser for all 294 records with per-record summaries, official conclusions and source links
- Seven-storyline deep analysis (WWII foo fighters → Cold War wave → Blue Book 14 → CIA U-2 → space program → sensor era → Western-facility cluster)
- Two games: Analyst Exam (explained vs unresolved, real cases) & Timeline Challenge
- Reference links (official portal, archives, mirrors, news)

Static site, zero dependencies. Summaries were produced by AI-assisted close reading of the
original files (OCR/whisper pipeline); they may contain errors — the original files at
war.gov/ufo are authoritative. Unofficial project.
